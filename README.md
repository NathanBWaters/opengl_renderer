# OpenGL Renderer
Made my own OpenGL Renderer using C++, XCode, Qt, glm, GLFW, and assimp.

Extremely fun side project.  It has some of the following features:
1) Point lights
2) Area lights
3) Texture rendering
4) Moveable camera
5) Particle system
6) .obj loader
